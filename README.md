# Nipuna Sachintha Portfolio

Personal portfolio website built with React and Vite.

## Live Website

https://sachinthax.live

## Tech Stack

- React
- Vite
- CSS
- Lenis
- Lucide React
- React Icons

## Features

- Responsive design
- Smooth scrolling
- Project showcase
- Skills section
- Education and certifications
- Downloadable CV
- Contact section

## Run Locally

```bash
git clone https://github.com/SachinthaX/sachinthax-portfolio.git
cd sachinthax-portfolio
npm install
npm run dev
```
